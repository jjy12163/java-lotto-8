# 기능 요구 사항
- 로또 번호의 숫자 범위 : 1~45
- 로또 1개당 중복되지 않는 숫자 6개
- 당첨 번호 추첨 : 중복되지 않는 숫자 6개와 보너스 번호 1개
- 당첨
    - 1등 : 번호 6개 일치 /  2,000,000,000원
    - 2등 : 번호 5개 일치 + 보너스 번호 일치 / 30,000,000원
    - 3등 : 번호 5개 일치 / 1,500,000원
    - 4등 : 번호 4개 일치 / 50,000원
    - 5등 : 번호 3개 일치 / 5,000원
- 로또 구입 금액 입력 - 해당 금액만큼 발행해줌(1장에 천원)
- 입력 : 구입 금액, 당첨 번호, 보너스 번호
- 잘못된 값 입력 - IllegalArgumentException 발생, “[ERROR]”로 시작하는 에러 메시지 출력 후 그 부분부터 입력 다시

# 입출력 요구 사항
입력 1 : 로또 구입 금액

입력 2 : 당첨 번호 6개 입력(쉼표(,)를 기준으로 구분)

입력 3 : 보너스 번호 입력

출력 1 : 발행한 로또 수량과 번호 출력(번호는 오름차순으로 정렬)

출력 2 : 당첨 내역 출력

출력 3 : 수익률(소수점 둘째 자리에서 반올림)

### 실행 결과 예시
```
구입금액을 입력해 주세요.
8000

8개를 구매했습니다.
[8, 21, 23, 41, 42, 43] 
[3, 5, 11, 16, 32, 38] 
[7, 11, 16, 35, 36, 44] 
[1, 8, 11, 31, 41, 42] 
[13, 14, 16, 38, 42, 45] 
[7, 11, 30, 40, 42, 43] 
[2, 13, 22, 32, 38, 45] 
[1, 3, 5, 14, 22, 45]

당첨 번호를 입력해 주세요.
1,2,3,4,5,6

보너스 번호를 입력해 주세요.
7

당첨 통계
---
3개 일치 (5,000원) - 1개
4개 일치 (50,000원) - 0개
5개 일치 (1,500,000원) - 0개
5개 일치, 보너스 볼 일치 (30,000,000원) - 0개
6개 일치 (2,000,000,000원) - 0개
총 수익률은 62.5%입니다.
```

**Exception : 예외 상황 시에는 에러 문구 출력 후 틀린 부분부터 다시 출력**
```
[ERROR] 로또 번호는 1부터 45 사이의 숫자여야 합니다.
```


# 간략한 플로우
1. 사용자가 로또 구입 금액을 입력
2. 로또 발행 횟수만큼 6개의 숫자로 이루어진 리스트 출력
3. 사용자가 당첨 번호 6개 , 보너스 번호 1개 입력
4. 2번에서 만들어진 숫자 리스트와 입력받은 당첨 번호 6개의 비교 - 일치하는 숫자 갯수 확인
5. 당첨 통계와 수익률(당첨 금액/구입 금액*100) 출력


# 예외 처리
### 구입 금액 입력
- 입력값이 숫자가 아닌 문자열 포함
  - Error Meessage : 구입 금액은 숫자로 입력해 주세요.
- 양의 정수가 아닌 다른 수 입력
  - Error Message : 구입 금액은 양의 정수로 입력해 주세요
- 1000으로 나누어 떨어지지 않는 숫자 입력
  - Error Message : 1000원 단위로 입력해 주세요.

### 로또 번호 입력
- 번호 형식이 숫자가 아닌 경우
  - Error Message : 로또 번호는 쉼표(,)로 구분된 숫자로 입력해 주세요.)
- 숫자 개수가 6개가 아닌 경우 혹은 빈칸이 있는 경우(,,  혹은 ,로 끝남)
  - Error Message : 숫자는 6개를 입력해 주셔야 합니다.
- 중복된 숫자를 입력한 경우
  - Error Message : 중복 없이 입력해 주세요.
- 1미만 혹은 45초과의 값 입력
  - Error Message : 1이상 45이하의 값만 입력해 주세요.

### 보너스 번호 입력
- 공백 입력
  - Error Message : 값을 입력해 주세요
- 번호 형식이 숫자가 아닌 경우
  - Error Message : 보너스 번호는 숫자를 입력해 주세요.
- 보너스 번호가 로또 번호와 중복될 경우
  - Error Message : 이전 입력값과 중복되지 않은 값을 입력해 주세요.
- 1미만 혹은 45초과의 값 입력
  - Error Message : 1이상 45이하의 값만 입력해 주세요.
